Number Theory Advanced IMO 2023 · P2
Find all integers n ≥ 1 such that the following system has a solution
Find all integers \( n \geq 1 \) such that there exist positive integers \( a_1, a_2, \ldots, a_n \) satisfying: $$a_1 + a_2 + \cdots + a_n = n^2$$ $$\frac{1}{a_1} + \frac{1}{a_2} + \cdots + \frac{1}{a_n} = 1$$

By AM-HM inequality applied to \(a_1, \ldots, a_n\):
\frac{a_1 + \cdots + a_n}{n} \geq \frac{n}{\frac{1}{a_1} + \cdots + \frac{1}{a_n}}
Substituting the given conditions \(\sum a_i = n^2\) and \(\sum \frac{1}{a_i} = 1\), we get equality. Equality holds in AM-HM iff all \(a_i\) are equal. So \(a_i = n\) for all \(i\). This works for all \(n \geq 1\). The answer is all positive integers \(n \geq 1\). ∎

Here's an alternate approach using Cauchy-Schwarz. By CS inequality:
\left(\sum_{i=1}^n a_i\right)\left(\sum_{i=1}^n \frac{1}{a_i}\right) \geq n^2
We have \(n^2 \cdot 1 = n^2\), so equality holds! Equality in CS requires all \(a_i\) equal, giving \(a_i = n\) for all \(i\). Clean and elegant!
